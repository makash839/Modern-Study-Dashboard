# 🎯 Study Target Dashboard with ARS Integration

## Features

✅ **Subject-wise Topic Tracking**
- Add multiple subjects
- Add topics under each subject
- Start and Finish tracking

✅ **Intelligent Time Calculation**
- Auto-calculates time based on your completion speed
- Shows estimated days to finish each subject and entire course

✅ **ARS (Active Recall System) Revision**
- Automatic revision scheduling: 1, 3, 7, 15, 30 days after completion
- Daily to-do list for revisions
- Track revision completion

✅ **Modern UI**
- Professional dashboard design
- Real-time progress tracking
- Beautiful charts and stats

## How to Use

1. Open `index.html` in your browser
2. Click "Add Subject" to create subjects
3. Add topics to each subject
4. Click "Start" when you begin a topic
5. Click "Finish" when you complete it
6. System will automatically schedule revisions
7. Check "Today's To-Do List" for daily revisions

## Algorithm Logic

**Time Estimation:**
- Tracks start and completion time for each topic
- Calculates average days per topic
- Multiplies by remaining topics for total estimate

**ARS Revision Schedule:**
- Day 1: First revision
- Day 3: Second revision
- Day 7: Third revision
- Day 15: Fourth revision
- Day 30: Final revision

## Installation

Just download all files in the same folder and open `index.html`!
